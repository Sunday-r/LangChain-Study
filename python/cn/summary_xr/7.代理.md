# LangChain：代理（Agents）— 代码笔记

核心思想：将 LLM 当作**推理引擎**，让它自主决定调用哪些工具来完成任务。代理循环执行"思考→行动→观察"，直到得出最终答案。

---

## 一、ReAct 循环（代理的工作方式）

```
Thought（思考）：分析问题，决定下一步做什么
    ↓
Action（行动）：选择一个工具并传入参数
    ↓
Observation（观察）：获取工具返回的结果
    ↓
重复以上步骤，直到 Thought 认为可以给出 Final Answer
```

---

## 二、使用内置工具：llm-math + wikipedia

### 2.1 初始化模型、加载工具、创建代理

```python
from langchain.agents import load_tools, initialize_agent, AgentType
from langchain.chat_models import ChatOpenAI

llm = ChatOpenAI(temperature=0)

# 加载内置工具
tools = load_tools(["llm-math", "wikipedia"], llm=llm)

# 初始化代理
agent = initialize_agent(
    tools,                                          # 工具列表
    llm,                                            # 语言模型
    agent=AgentType.CHAT_ZERO_SHOT_REACT_DESCRIPTION,  # 代理类型
    handle_parsing_errors=True,                     # 解析错误时让 LLM 自纠
    verbose=True,                                   # 打印中间步骤
)
```

**要点：**
- `CHAT_ZERO_SHOT_REACT_DESCRIPTION`：CHAT 表示用对话模型，REACT 表示使用 ReAct 提示模板，ZERO_SHOT 表示无需示例
- `handle_parsing_errors=True`：输出格式出错时将错误信息反馈给 LLM 重新生成

### 2.2 数学问题（使用 llm-math 工具）

```python
agent("What is the 25% of 300?")
```

**内部过程：**
```
Thought: 需要计算 300 的 25%，用到乘法和除法
Action: Calculator, 输入 "300*0.25"
Observation: 75.0
Thought: 已得到答案
Final Answer: 75.0
```

### 2.3 知识查询（使用 wikipedia 工具）

```python
question = "Tom M. Mitchell is an American computer scientist \
and the Founders University Professor at Carnegie Mellon University (CMU)\
what book did he write?"
agent(question)
```

**内部过程：**
```
Thought: 应该使用 Wikipedia 搜索
Action: Wikipedia, 输入 "Tom M. Mitchell"
Observation: Tom Michael Mitchell... author of the textbook Machine Learning
Thought: Tom M. Mitchell 写的书是 Machine Learning
Final Answer: Machine Learning.
```

## 三、使用 PythonREPLTool 工具

让代理能执行 Python 代码。

```python
from langchain.agents.agent_toolkits import create_python_agent
from langchain.tools.python.tool import PythonREPLTool

# 创建 Python 代理（一步完成）
agent = create_python_agent(
    llm,
    tool=PythonREPLTool(),
    verbose=True,
)

# 让代理写代码排序
customer_list = [["Harrison", "Chase"], ["Lang", "Chain"],
                 ["Dolly", "Too"], ["Elle", "Elem"]]

agent.run(f"Sort these customers by last name and then first name: {customer_list}")
```

**内部过程：**
```
Thought: 使用 sorted() 函数，key 为 (姓, 名)
Action: Python REPL
Action Input:
  sorted_customers = sorted(customers, key=lambda x: (x[1], x[0]))
  print(sorted_customers)
Observation: [['Elle', 'Elem'], ['Lang', 'Chain'], ...]
Thought: 排序完成
Final Answer: [['Elle', 'Elem'], ['Lang', 'Chain'], ...]
```

**要点：** 代理**自动编写并执行 Python 代码**，适合需要精确计算或数据处理的任务。

## 四、定义自定义工具

### 4.1 用 @tool 装饰器创建工具

```python
from langchain.agents import tool
from datetime import date

@tool
def time(text: str) -> str:
    """Returns todays date, use this for any \
    questions related to knowing todays date. \
    The input should always be an empty string, \
    and this function will always return todays \
    date - any date mathmatics should occur \
    outside this function."""
    return str(date.today())
```

**要点：**
- `@tool` 装饰器将普通函数转为 LangChain 工具
- **文档字符串非常重要**——代理通过阅读描述来决定何时、如何使用该工具
- 描述要写清楚：功能是什么、输入格式是什么、输出是什么

### 4.2 将自定义工具加入代理

```python
agent = initialize_agent(
    tools + [time],    # 将自定义工具追加到已有工具列表
    llm,
    agent=AgentType.CHAT_ZERO_SHOT_REACT_DESCRIPTION,
    handle_parsing_errors=True,
    verbose=True,
)

agent("whats the date today?")
```

**内部过程：**
```
Thought: 需要使用 time 工具获取今天的日期
Action: time, 输入 ""
Observation: 2023-06-03
Thought: 成功获取日期
Final Answer: Today's date is 2023-06-03.
```

## 五、调试代理：langchain.debug

```python
import langchain

langchain.debug = True   # 开启：显示完整的链调用、LLM 输入输出、工具调用细节
agent.run(...)
langchain.debug = False  # 关闭
```

debug 模式下可以看到：
- 发送给 LLM 的完整 prompt（包含工具描述和 ReAct 格式指令）
- LLM 每次返回的 Thought/Action
- 工具的输入和输出
- 每步的耗时

## 六、关键 API 速查

| API | 作用 |
|---|---|
| `load_tools(["llm-math", "wikipedia"], llm=llm)` | 加载内置工具 |
| `initialize_agent(tools, llm, agent=..., verbose=True)` | 创建通用代理 |
| `create_python_agent(llm, tool=PythonREPLTool())` | 快捷创建 Python 代理 |
| `@tool` 装饰器 | 将函数转为 LangChain 工具 |
| `agent("问题")` / `agent.run("问题")` | 运行代理 |
| `AgentType.CHAT_ZERO_SHOT_REACT_DESCRIPTION` | 常用代理类型（对话模型 + ReAct） |
| `langchain.debug = True` | 调试模式 |

## 七、三种工具对比

| 工具 | 用途 | 特点 |
|---|---|---|
| `llm-math` (Calculator) | 数学计算 | LLM 生成算式，计算器执行 |
| `wikipedia` | 知识查询 | 通过 API 搜索维基百科 |
| `PythonREPLTool` | 执行 Python 代码 | 代理自动写代码并运行 |
| `@tool` 自定义工具 | 任意功能 | 开发者自定义，描述决定代理调用时机 |
