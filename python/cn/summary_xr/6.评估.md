# LangChain：评估（Evaluation）— 代码笔记

核心思想：**用 LLM 来评估 LLM**。对基于文档问答的应用，生成测试数据 → 运行链得到预测 → 用 LLM 判断预测是否正确。

---

## 一、构建待评估的 QA 应用

复用上一章的 RetrievalQA 链：

```python
from langchain.chains import RetrievalQA
from langchain.chat_models import ChatOpenAI
from langchain.document_loaders import CSVLoader
from langchain.indexes import VectorstoreIndexCreator
from langchain.vectorstores import DocArrayInMemorySearch

# 加载文档，创建向量索引
file = 'OutdoorClothingCatalog_1000.csv'
loader = CSVLoader(file_path=file)
index = VectorstoreIndexCreator(
    vectorstore_cls=DocArrayInMemorySearch
).from_loaders([loader])

# 创建检索问答链
llm = ChatOpenAI(temperature=0.0)
qa = RetrievalQA.from_chain_type(
    llm=llm,
    chain_type="stuff",
    retriever=index.vectorstore.as_retriever(),
    verbose=True,
    chain_type_kwargs={
        "document_separator": "<<<<>>>>>"  # 文档之间的分隔符
    }
)
```

## 二、创建测试数据（两种方式）

### 2.1 手动构造测试用例

```python
examples = [
    {
        "query": "Do the Cozy Comfort Pullover Set have side pockets?",
        "answer": "Yes"
    },
    {
        "query": "What collection is the Ultra-Lofty 850 Stretch Down Hooded Jacket from?",
        "answer": "The DownTek collection"
    }
]
```

**要点：** 人工查看文档内容，编写问题和标准答案。

### 2.2 用 LLM 自动生成测试用例

```python
from langchain.evaluation.qa import QAGenerateChain

# 创建 QA 生成链
example_gen_chain = QAGenerateChain.from_llm(ChatOpenAI())

# 传入文档，自动生成问答对
new_examples = example_gen_chain.apply_and_parse(
    [{"doc": t} for t in data[:5]]
)

# 合并到测试集
examples += new_examples
```

**要点：** `QAGenerateChain` 接收文档，自动为每个文档生成一对 question + answer，省去人工编写。

## 三、人工调试：langchain.debug

```python
import langchain

# 开启调试模式，查看链执行的每一步细节
langchain.debug = True
qa.run(examples[0]["query"])

# 关闭调试
langchain.debug = False
```

**debug 输出包含：**
- 检索到了哪些文档（context）
- 发送给 LLM 的完整 prompt
- LLM 返回的原始结果
- token 使用量

**要点：** 当问答结果出错时，问题通常出在**检索阶段**（找错了文档），而非 LLM 本身。debug 模式帮你定位是检索问题还是生成问题。

## 四、LLM 自动评估

### 4.1 批量运行得到预测结果

```python
# 对所有测试用例运行 QA 链，得到预测结果
predictions = qa.apply(examples)
```

### 4.2 用 LLM 评估预测是否正确

```python
from langchain.evaluation.qa import QAEvalChain

# 创建评估链
eval_chain = QAEvalChain.from_llm(ChatOpenAI(temperature=0))

# 评估：对比 examples（标准答案）和 predictions（预测答案）
graded_outputs = eval_chain.evaluate(examples, predictions)
```

### 4.3 查看评估结果

```python
for i, eg in enumerate(examples):
    print(f"Example {i}:")
    print("Question: " + predictions[i]['query'])
    print("Real Answer: " + predictions[i]['answer'])
    print("Predicted Answer: " + predictions[i]['result'])
    print("Predicted Grade: " + graded_outputs[i]['text'])  # CORRECT 或 INCORRECT
    print()
```

**输出示例：**
```
Example 0:
Question: Do the Cozy Comfort Pullover Set have side pockets?
Real Answer: Yes
Predicted Answer: The Cozy Comfort Pullover Set, Stripe does have side pockets.
Predicted Grade: CORRECT
```

**要点：** 评估链不要求精确字符串匹配，而是用 LLM 判断语义是否一致。比如 "Yes" 和 "does have side pockets" 会被判定为 CORRECT。

## 五、整体评估流程

```
文档数据
   ↓
┌─────────────────────────────────┐
│  准备测试数据                     │
│  ① 手动编写 Q&A 对               │
│  ② QAGenerateChain 自动生成 Q&A  │
└─────────────────────────────────┘
   ↓
examples（标准问答对）
   ↓
qa.apply(examples) → predictions（预测结果）
   ↓
QAEvalChain.evaluate(examples, predictions) → graded_outputs（评分）
   ↓
遍历输出：Question / Real Answer / Predicted Answer / Grade
```

## 六、关键 API 速查

| API | 作用 |
|---|---|
| `QAGenerateChain.from_llm(llm)` | 用 LLM 从文档自动生成测试问答对 |
| `qa.apply(examples)` | 批量运行 QA 链，得到所有预测结果 |
| `QAEvalChain.from_llm(llm)` | 创建评估链，用 LLM 判断预测是否正确 |
| `eval_chain.evaluate(examples, predictions)` | 执行评估，返回 CORRECT / INCORRECT |
| `langchain.debug = True` | 开启调试，查看链执行的中间步骤 |
