## 一、直接调用 OpenAI API（原始写法）

```python
import openai

def get_completion(prompt, model="gpt-3.5-turbo"):
    messages = [{"role": "user", "content": prompt}]
    response = openai.ChatCompletion.create(
        model=model,
        messages=messages,
        temperature=0,
    )
    return response.choices[0].message["content"]
```

**要点：** 手动拼 prompt → 手动构造 messages → 手动提取结果。问题：不可复用，输出是纯字符串。

`get_completion`函数是基于`openai`的封装函数，对于给定提示（prompt）输出相应的回答。其包含两个参数
   
   - `prompt` 必需输入参数。 你给模型的提示，可以是一个问题，可以是你需要模型帮助你做的事（改变文本写作风格，翻译，回复消息等等）。
   - `model` 非必需输入参数。默认使用gpt-3.5-turbo。你也可以选择其他模型。
   
这里的提示对应我们给chatgpt的问题，函数给出的输出则对应chatpgt给我们的答案。

## 二、LangChain 模型

```python
from langchain.chat_models import ChatOpenAI
# 这里我们将参数temperature设置为0.0，从而减少生成答案的随机性。
# 如果你想要每次得到不一样的有新意的答案，可以尝试调整该参数。
# temperature越高 随机性越强

chat = ChatOpenAI(temperature=0.0)

```

**要点：** 一行创建模型实例，`temperature=0` 减少随机性。之后直接 `chat(messages)` 调用。

## 三、提示模板（替代 f-string）

**步骤：构造模板 → 创建模板对象 → 填充变量 → 调用模型**

```python
from langchain.prompts import ChatPromptTemplate

# 1. 写模板字符串，用 {变量名} 占位
template_string = """Translate the text that is delimited by triple backticks
into a style that is {style}.
text: ```{text}```
"""

# 2. 转为模板对象（自动识别出 style、text 两个变量）
prompt_template = ChatPromptTemplate.from_template(template_string)

# 3. 填充变量，生成消息列表
customer_messages = prompt_template.format_messages(
    style="American English in a calm and respectful tone",
    text=customer_email,
)

# 4. 调用模型
customer_response = chat(customer_messages)
print(customer_response.content)
```

**对比原始写法的好处：**

- 模板可复用，换一组变量就能生成新提示
- 变量名自动提取，不容易拼错
- 输出是结构化的 `HumanMessage` 对象，不是裸字符串

## 四、输出解析器（让模型输出可编程处理）

**问题：** 模型输出是 JSON 字符串，`response.content` 是 `str`，不能用 `.get()` 取值。

**解决步骤：**

```python
from langchain.output_parsers import ResponseSchema, StructuredOutputParser

# 1. 定义每个字段的 schema
gift_schema = ResponseSchema(name="gift", description="是否是礼物？True/False")
delivery_days_schema = ResponseSchema(name="delivery_days", description="发货天数，未知填-1")
price_value_schema = ResponseSchema(name="price_value", description="价格相关句子，逗号分隔列表")

# 2. 创建解析器，自动生成格式化指令
output_parser = StructuredOutputParser.from_response_schemas(
    [gift_schema, delivery_days_schema, price_value_schema]
)
format_instructions = output_parser.get_format_instructions()

# 3. 模板中加入 {format_instructions} 占位符
review_template_2 = """...text: {text}
{format_instructions}"""
prompt = ChatPromptTemplate.from_template(review_template_2)

# 4. 填充并调用
messages = prompt.format_messages(text=customer_review, format_instructions=format_instructions)
response = chat(messages)

# 5. 解析输出 → 得到 dict
output_dict = output_parser.parse(response.content)
output_dict.get("gift")           # True
output_dict.get("delivery_days")  # "2"
```

**关键流程：** `ResponseSchema` 定义字段 → `StructuredOutputParser` 生成格式指令 → 指令注入提示 → 模型按格式输出 → `parse()` 转为字典。

---

## 五、整体调用链路

```
模板字符串 → ChatPromptTemplate → format_messages(变量) → chat(messages) → .content
                                                          ↓
                                        需要结构化输出时：
                                        ResponseSchema → StructuredOutputParser
                                        → get_format_instructions() 注入模板
                                        → parse(response.content) → dict
```
