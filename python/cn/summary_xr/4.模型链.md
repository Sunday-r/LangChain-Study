# LangChain：Chains（链）— 代码笔记

链（Chains）将多个组件（LLM + Prompt）组合在一起，对文本进行一系列操作。本节介绍三种链。

---

## 一、LLMChain（基础链）

**最简单的链 = Prompt + LLM，也是后续所有链的基础。**

```python
from langchain.chat_models import ChatOpenAI
from langchain.prompts import ChatPromptTemplate
from langchain.chains import LLMChain

llm = ChatOpenAI(temperature=0.9)

prompt = ChatPromptTemplate.from_template(
    "What is the best name to describe a company that makes {product}?"
)

# 将 llm 和 prompt 组合成链
chain = LLMChain(llm=llm, prompt=prompt)

# 运行链
product = "Queen Size Sheet Set"
chain.run(product)  # → 'Regal Beddings.'
```

**要点：** `LLMChain(llm, prompt)` 组合模型和模板，`chain.run(变量值)` 一行执行。

---

## 二、Sequential Chains（顺序链）

将多个子链**串联**，前一个的输出作为后一个的输入。

### 2.1 SimpleSequentialChain

**适用场景：** 每个子链只有单个输入和单个输出，严格串行。

```python
from langchain.chains import SimpleSequentialChain

# 子链1：产品 → 公司名称
first_prompt = ChatPromptTemplate.from_template(
    "What is the best name to describe a company that makes {product}?"
)
chain_one = LLMChain(llm=llm, prompt=first_prompt)

# 子链2：公司名称 → 公司描述
second_prompt = ChatPromptTemplate.from_template(
    "Write a 20 words description for the following company:{company_name}"
)
chain_two = LLMChain(llm=llm, prompt=second_prompt)

# 组合
overall_simple_chain = SimpleSequentialChain(
    chains=[chain_one, chain_two],
    verbose=True,  # 可查看中间过程
)

overall_simple_chain.run("Queen Size Sheet Set")
# 输出：公司描述（由子链1生成名称 → 子链2生成描述）
```

**要点：** `chains` 列表中顺序即为执行顺序，**上一个输出自动作为下一个输入**。

### 2.2 SequentialChain

**适用场景：** 多个输入/输出，子链之间可以**并行或跳步**（不严格串行）。

```python
from langchain.chains import SequentialChain

# 子链1：Review → 英文翻译
chain_one = LLMChain(
    llm=llm,
    prompt=ChatPromptTemplate.from_template(
        "Translate the following review to english:\n\n{Review}"
    ),
    output_key="English_Review",  # 指定输出变量名
)

# 子链2：英文Review → 一句话总结
chain_two = LLMChain(
    llm=llm,
    prompt=ChatPromptTemplate.from_template(
        "Can you summarize the following review in 1 sentence:\n\n{English_Review}"
    ),
    output_key="summary",
)

# 子链3：Review → 识别语言
chain_three = LLMChain(
    llm=llm,
    prompt=ChatPromptTemplate.from_template(
        "What language is the following review:\n\n{Review}"
    ),
    output_key="language",
)

# 子链4：总结 + 语言 → 用原文语言写后续回复
chain_four = LLMChain(
    llm=llm,
    prompt=ChatPromptTemplate.from_template(
        "Write a follow up response to the following summary "
        "in the specified language:\n\nSummary: {summary}\n\nLanguage: {language}"
    ),
    output_key="followup_message",
)

# 组合
overall_chain = SequentialChain(
    chains=[chain_one, chain_two, chain_three, chain_four],
    input_variables=["Review"],                                      # 整体输入
    output_variables=["English_Review", "summary", "followup_message"],  # 整体输出
    verbose=False,
)

# 运行
overall_chain(df.Review[5])
```

**要点：**
- 每个子链通过 `output_key` 命名输出，后续子链通过变量名引用
- 子链3 直接用原始 `Review`，不依赖子链1 → 体现了非严格串行
- `input_variables` / `output_variables` 声明整体链的输入和最终要保留的输出

## 三、Router Chain（路由链）

**根据输入内容，自动选择不同的子链处理。** 类似于 if-else 路由分发。

### 整体流程

```
定义多个提示模板 → 为每个模板命名+描述
→ 创建目标链（每个模板一个 LLMChain）
→ 创建路由链（LLM 根据描述自动选择）
→ 创建默认链（兜底）
→ MultiPromptChain 组合
```

### 代码

```python
from langchain.chains.router import MultiPromptChain
from langchain.chains.router.llm_router import LLMRouterChain, RouterOutputParser
from langchain.prompts import PromptTemplate

# 1. 定义多个提示模板（各领域的专家人设）
physics_template = """You are a very smart physics professor...
Here is a question: {input}"""

math_template = """You are a very good mathematician...
Here is a question: {input}"""

history_template = """You are a very good historian...
Here is a question: {input}"""

computerscience_template = """You are a successful computer scientist...
Here is a question: {input}"""

# 2. 为每个模板命名并提供描述（路由链据此选择）
prompt_infos = [
    {"name": "physics", "description": "Good for answering questions about physics",
     "prompt_template": physics_template},
    {"name": "math", "description": "Good for answering math questions",
     "prompt_template": math_template},
    {"name": "History", "description": "Good for answering history questions",
     "prompt_template": history_template},
    {"name": "computer science", "description": "Good for answering computer science questions",
     "prompt_template": computerscience_template},
]

# 3. 创建目标链（每个领域一条）
destination_chains = {}
for p_info in prompt_infos:
    prompt = ChatPromptTemplate.from_template(p_info["prompt_template"])
    chain = LLMChain(llm=llm, prompt=prompt)
    destination_chains[p_info["name"]] = chain

destinations_str = "\n".join(
    [f"{p['name']}: {p['description']}" for p in prompt_infos]
)

# 4. 创建默认链（兜底，当路由器无法匹配时使用）
default_chain = LLMChain(
    llm=llm,
    prompt=ChatPromptTemplate.from_template("{input}")
)

# 5. 构建路由链（LLM 根据描述自动判断路由到哪条子链）
router_template = MULTI_PROMPT_ROUTER_TEMPLATE.format(destinations=destinations_str)
router_prompt = PromptTemplate(
    template=router_template,
    input_variables=["input"],
    output_parser=RouterOutputParser(),
)
router_chain = LLMRouterChain.from_llm(llm, router_prompt)

# 6. 组合成多提示链
chain = MultiPromptChain(
    router_chain=router_chain,
    destination_chains=destination_chains,
    default_chain=default_chain,
    verbose=True,
)

# 7. 运行（自动路由到对应领域链）
chain.run("What is black body radiation?")  # → 路由到 physics 链
chain.run("what is 2 + 2")                  # → 路由到 math 链
chain.run("你知道李白是谁嘛?")               # → 路由到 History 链
```

**要点：**
- `description` 是路由的关键——LLM 据此判断输入属于哪个领域
- `RouterOutputParser` 解析路由链的输出，提取目标链名称
- 当输入不匹配任何已有链时，应路由到 `DEFAULT`（默认链兜底）
- `MULTI_PROMPT_ROUTER_TEMPLATE` 是 LangChain 内置的路由提示模板，格式固定

---

## 四、三种链对比总结

| 链类型 | 适用场景 | 关键特点 |
|---|---|---|
| `LLMChain` | 单步 Prompt + LLM | 最基础的链，所有链的构建块 |
| `SimpleSequentialChain` | 单输入→单输出的流水线 | 严格串行，上一步输出 = 下一步输入 |
| `SequentialChain` | 多输入多输出的流水线 | 通过 `output_key` 连接，可并行/跳步 |
| `Router Chain` | 根据输入内容分发到不同子链 | LLM 自动路由，需要描述信息 + 默认链兜底 |
